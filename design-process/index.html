Design Process

My design process depends on various factors regarding the client and the project.

Design Thinking

  1. Empathize and Discover
  2. Define
  3. Ideate
  4. Prototype
  5. Test
  6. Implement

This is a research and design process. Its used at the beginning of a project to discover the best design solution for a project.

Tree

This is a research and implementation process for redesigning existing projects.

It examines the root problem, past research, and the current situation to develop an informed perspective. This is used to create new user flows and interfaces that improve the project.

Fulfillment

This is used when the client has completed user experience research and written a style guide for their project.

In that situation, they've already done user interviews and defined the fonts, colors, component styles, and sizes.
They need a designer to follow their plan and complete the screen designs or templates for the project.
