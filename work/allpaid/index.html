AllPaid

 

UX/UI DESIGNER

I worked for Aventiv on the AllPaid Portal Project where I collaborated with the project manager and developers to complete screen designs for backlogged JIRA tickets.
 

Problem

The AllPaid Payment Portal provides online payment processing services to local and state governments.

The project was already a year in production with many new features on the road map. The colors, fonts, styles, and layouts had already been decided. The previous designer wasn't able to keep up with the workload. The bottleneck was costing time and money.

They needed a designer who could go through the backlogged JIRA tickets and complete the planned responsive screen designs for the developers.

Solution

To complete the backlogged JIRA tickets quickly, I built an assembly line process with reusable components and templates.

I worked with the project manager and used Trello to create a visual inventory of the most urgent tickets to work on.

In collaboration with the project manager and the developers we completed the backlogged tickets and implemented the following features.

  • AllPaid Wallet allowed users to store up to five different payment methods.
  • Shopping Cart allowed users to add multiple items to a cart and pay the total with one payment method.
  • Autopay gave users the option to setup a default payment method for monthly payments.
