HCCMIS

 

UI Developer

I worked for HCCMIS on the Atlas Travel Project l where I wrote CSS and Bootstrap code to create responsive web pages.

 

Problem

The company offers short-term medical travel insurance to customers who are vacationing outside the United States.

The Atlas Travel Project was a refresh of their main insurance product and their website.

They wanted their redesigned website to be responsive and mobile-friendly.

Solution

I was contracted to convert Photoshop web page mockups into responsive HTML and Bootstrap CSS code.

I collaborated with the graphic designer to streamline the design hand off process. Then wrote code to complete each page as requested.

My manager tested and deployed the pages on the company's PHP-based Cascade CMS server.
