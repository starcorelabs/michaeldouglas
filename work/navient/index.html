Navient

 

Role

I worked as a UI Developer for Navient on the Loan Repayment Portal where I wrote CSS and Bootstrap code to create responsive front-end views.

 

Challenge

A team of back-end C# developers working offshore in India wrote buggy front-end HTML and Bootstrap CSS code for the web pages.

This caused the pages to look and behave badly on all devices.

Solution

My 12 month contract as a UI Developer consisted of fixing bugs in the Bootstrap layouts that were created by the offshore back-end developers.

I restructured the HTML and rewrote CSS code to make the web pages adapt to all devices. While also passing QA requirements and 508 compliance.
