BlueSky Commerce

 

Role

I worked as a UI Consultant at BlueSky Commerce on the Midwest Veterinary Supply project. My role focused on writing CSS code that helped web pages adjust smoothly to different screen sizes and devices. This ensured the site looked clean, readable, and easy to use on desktops, tablets, and mobile phones.

 

Challenge

Midwest Veterinary Supply (MVS) is like the Amazon of the veterinary world. It provides an online system that helps veterinary clinics manage their entire business.

MVS wanted to redesign its eCommerce platform and add an admin area. This new system would help clinics manage staff and supplies, while also allowing pet owners to create accounts and schedule visits.

When I joined the team, the project was already about 75% complete. They needed a UI Developer to finish existing JIRA tickets and fix front-end web pages that did not adjust well to different screen sizes.

I worked on a laptop using a required VMWare virtual environment. The development tools inside the virtual machine often froze or showed errors, which slowed progress and required extra troubleshooting.

At the same time, many JIRA tickets were unclear and had strict time limits. This meant I had to ask for more details before starting, since clear goals are needed to complete work correctly.

Despite these challenges, I stayed focused, communicated clearly, and worked through each issue to keep the project moving forward.

Solution

To solve these challenges, I worked closely with my manager to update and improve the custom HTML and CSS. My goal was to make sure every page was responsive, mobile-friendly, and met quality standards.

I used Photoshop design files to carefully match the look and feel of each page. I also followed written guidelines to make sure all labels and text were correct and easy to understand.

To stay aligned with the team, I joined daily meetings to keep up with changing priorities and project updates. Near the end of the project, our team successfully shifted to working from home during the pandemic and continued delivering progress without interruption.
